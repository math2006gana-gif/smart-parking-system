# smart-parking-system
AI-powered license plate recognition system
## Features
- Automatic license plate detection
- Real-time OCR
- Entry/Exit tracking
- Works in dark/angled conditions
- 90-95% accuracy

## Tech Stack
- Python Flask
- EasyOCR
- OpenCV
- MySQL

## Installation
```bash
pip install -r requirements.txt
python app.py
```

## Usage
Open http://127.0.0.1:5000 in browser
